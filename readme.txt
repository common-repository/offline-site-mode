== Installation == 
#### Thank you for your interest in Offline Site.  
### Minimum requirements.  

* Wordpress 3.0+ 
* PHP 5.x 
* MySQL 5.x  

This section describes how to install the plugin and get it working.
e.g. 
 
1. Upload folder 'plugin-name' to the `/wp-content/plugins/` directory.
2. Activate the "Offline Site" plugin through the 'Plugins' menu in WordPress.
3. Go to "Offline Site" in Setting.
